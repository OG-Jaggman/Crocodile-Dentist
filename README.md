# Crocodile Dentist

A simple desktop Crocodile Dentist game built with Python and `tkinter`, using `Crocodile-Dentist.png` as the crocodile model.

## Run it

```powershell
python crocodile_dentist.py
```

## How to play

- Click one of the crocodile's teeth each turn.
- Most teeth are safe.
- One random tooth snaps the crocodile's mouth shut and ends the round.
- Press `New Game` to reset and try again.
